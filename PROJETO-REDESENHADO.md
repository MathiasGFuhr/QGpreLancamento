## 🔧 **COMO USAR O PROJETO REDESENHADO**

### **1. Configuração Rápida**
```bash
# Execute o script de instalação
./install.sh

# Configure as credenciais do Supabase
nano js/config-simple.js
```

### **2. Para Desenvolvimento**
- Abra `index.html` em um servidor local
- Use Live Server no VS Code
- Ou `python -m http.server 8000`

### **3. Para Produção**
- Faça upload para seu servidor
- Configure SSL/HTTPS
- O arquivo `.htaccess` já está pronto
